--- Instruction to Install the Flutter SDK for development on windows ---

Download the Flutter SDK - ~400MB Place it in the C drive (OPTIONAL)
https://docs.flutter.dev/get-started/install/windows
(Instruction to completly setting up the development environment is given in the Above link).

Set the Path in environment vairables

Download the Android Studio (Windows/ MAC) / XCode for IOS development on Mac computers. 

Install the command line tools to avoid the warning message in flutter doctor.

Set a up a Emulator.

Install IntelliJ and Install Flutter Plugin and Dart Plugin.

When creating a project select the SDK path if not automatically selected by IntelliJ.
